
# Nen Notebook — Working Site Starter

This is a real static multi-page site starter built with plain HTML, CSS, and JavaScript.

## What works right now
- Home page
- About page
- Fics archive with JS filters
- Journal page
- Guestbook demo using localStorage
- Links / asset bank page
- Responsive mobile nav toggle
- Reusable asset folder structure

## Important
The site is fully usable locally as a static site.

The **guestbook is a working demo**, but public submissions need a real backend or form service.
Good options:
- Netlify Forms
- Formspree
- Supabase
- Firebase
- Custom server/API

## Fastest way to put it online
1. Upload the folder to GitHub
2. Deploy with Netlify or GitHub Pages
3. Connect the guestbook form to a form backend
4. Replace placeholder links and starter SVGs with custom assets
